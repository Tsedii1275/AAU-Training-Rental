SYSTEM ADMINISTRATOR 

Key Features

User Management:
• Create, edit, activate/deactivate, and reset passwords for users.
• Role-based access: ICT Administrator, Campus Administrator.
• Search, filter, and paginate user lists for easy navigation.

Task & Resource Tracking:
• Monitor tasks, requests, and resource usage across departments.
• Detailed reporting and analytics to improve efficiency.

Responsive Dashboard:
• Clean and intuitive interface for desktop and mobile.
• Interactive components like modals, tables, and filters for seamless workflow.

Security & Permissions:
• Role-based access control ensures secure operations.
• Password reset and account activation/deactivation managed by administrators.

Tech Stack

Frontend: React, TypeScript, Shadcn UI components

State Management: Redux Toolkit / RTK Query
